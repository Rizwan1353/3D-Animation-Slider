# 3D Animation Slider

This project showcases a 3D rotating image slider built with **HTML** and **CSS**.  
It uses `transform-style: preserve-3d`, `perspective`, and keyframe animations to create a carousel effect where images orbit around the center. The design includes:

- A full-screen banner with background image support
- Smooth auto-rotation using CSS animations
- Responsive layout with centered positioning
- Customizable number of images via CSS variables (`--quantity`, `--position`)

The slider is lightweight, requires no JavaScript, and can be easily adapted for portfolios, galleries, or creative landing pages.
